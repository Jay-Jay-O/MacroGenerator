- Windows 11

- Python
  - *If not installed, install it by using **[this link](https://www.python.org/downloads/)***
 
## Current Features üõ†Ô∏è

- Action Loop
  - perform actions repeatedly by increasing the number in the *Loops* textbox
  - Add delays between actions
 
- Action List
  - Preview actions and click and drag to change their sequence
  - Edit and delete action/s however you like

- Add mouse action
  - Add a left click, right click, middle mouse click and a scroll action
  - Press **CTRL** to locate current mouse location
    - Start position when *drag checkbox* is ticked
  - Press **SHIFT** when the *drag checkbox* is ticked to locate secondary mouse location
    - End position when *drag checkbox* is ticked
  - Increase / decrease hold duration to simulate long presses and quick taps
